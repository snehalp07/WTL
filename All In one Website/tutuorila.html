PYTHON

Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured, object-oriented and functional programming. Wikipedia
Designed by: Guido van Rossum
First appeared: 20 February 1991; 31 years ago
OS: Windows, macOS, Linux/UNIX, Android and more
Filename extensions: .py,.pyi,.pyc,.pyd,.pyw,.pyz (since 3.5),.pyo (prior to 3.5)
Paradigm: Multi-paradigm: object-oriented, procedural (imperative), functional, structured, reflective
Preview release: 3.12.0a5 / 7 February 2023; 2 days ago
? Parent company: Python Software Foundation

JAVA

Java is a high-level, general-purpose programming language known for its platform independence and object-oriented approach.
It was designed to have as few implementation dependencies as possible, making it suitable for cross-platform development.
Java is widely used for developing mobile apps, web applications, and enterprise software.
Designed by: James Gosling
First appeared: 23 May 1995; 26 years ago
OS: Cross-platform
Filename extensions:.java, .class
Paradigm: Object-oriented, structured, imperative, generic, reflective
Preview release: Java 17 (17.0.2) / 14 December 2021; 3 months ago
? Parent company: Oracle Corporation (originally developed by Sun Microsystems)
